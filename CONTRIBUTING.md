# Contributing

Thanks for helping improve TF.ai.

## Development

```powershell
flutter pub get
flutter analyze
flutter test
```

For Windows desktop:

```powershell
flutter run -d windows
flutter build windows
```

## Pull Requests

- Keep changes focused.
- Run `flutter analyze` and `flutter test`.
- Do not commit build output, private settings, API keys, or local chat data.
- Do not add copyrighted assets unless they are licensed for redistribution.
