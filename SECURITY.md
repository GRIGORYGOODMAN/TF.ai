# Security

## Supported Versions

The current `main` branch is the only supported development line.

## Reporting Issues

Open a GitHub issue for security or privacy concerns. Do not include API keys, private chat logs, or personal data in public reports.

## API Keys

TF.ai currently stores API keys locally through app preferences. This is convenient for early desktop builds, but encrypted key storage is planned for a future release.

## Model Output

AI model output can be inaccurate or unsafe. Users are responsible for choosing providers, models, prompts, and content policies that match their own use case.
